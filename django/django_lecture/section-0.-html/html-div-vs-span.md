# HTML - Div vs Span

* div: html 코드를 그룹으로 나눌 때, 사용한다.
* span: 인라인 블럭을 나눌 때, 사용한다.



```html
<!DOCTYPE html>
<html lang="en">
<!-- metadata, title -->
<head>
  <meta charset="UTF-8">
  <title>This is title</title>
</head>
<!-- contents -->
<body> 
  <h1>Div vs Span</h1>
  
  <div>
    <h2>Group 1</h2>
    <p>This is Group 1</p>
  </div>

  <div>
    <h2>Group 2</h2>
    <p>This is Group 2</p>
  </div>
</body>
</html>
```
